# Hello (Go)

A minimal Go "Hello, world!" example.

## Run

- Using the Go tool (no build artifact):

  ```bash
  go run main.go
  ```

- Build and run:

  ```bash
  go build -o hello
  ./hello        # on WSL / macOS / Linux
  .\hello.exe   # on Windows PowerShell/CMD
  ```

Expected output:

```
Hello, world!
```
